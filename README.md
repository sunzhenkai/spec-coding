## 🎯 什么是 Spec Coding？

Spec Coding（规格驱动编程）是一种通过编写详细的规格说明（Specification）来指导代码开发的编程方法。当与 AI 助手配合使用时，清晰、完整的规格说明能够显著提高代码生成的质量和开发效率。

## 🚀 快速开始

1. 阅读 [AI Spec Coding 规约指南](./specs/ai-spec-coding-guide.md)
2. 使用提供的模板编写你的第一个规格说明
3. 与 AI 助手协作生成高质量代码
